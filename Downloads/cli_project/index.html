MCP Project

cli_project — MCP Server & Client in Python

A hands-on implementation of Anthropic's Model Context Protocol (MCP), covering all three core primitives — Tools, Resources, and Prompts — and how they connect Claude AI to external services.

View on GitHub Course Reference
The 3 MCP Primitives
Tools
Model-controlled. Functions Claude invokes to take actions or fetch data.
Resources
App-controlled. Read-only data via static or templated URIs.
Prompts
User-controlled. Pre-crafted templates for common AI workflows.
